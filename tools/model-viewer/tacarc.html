Reverse Engineering Notes

The cone uses two sets of texture coordinates.
Base UV1 wraps radially:
UV1 = (0.1, V_radial)
V_radial (outer) = 0.0625
V_radial (origin) = 0.9
Effect UV2 maps angularly and scrolls over time:
UV2 = (U_angular + offset, V_radial)
U_angular += 1.0 / (segments * 5)
offset += deltaTime * 0.1 * speed
Blended Additively in pixel shader:
C = 2.0 * StateColor * (t1.a*t1 + t0.a*t0)
Z-Fail Pass displays blocked regions in dark purple:
MatZFailColor = rgba(13, 0, 51, 90)

      ℹ Prop collision (.cache files) not yet implemented

      Level JSON references assets/extracted/physics/*.cache — Havok 2.3 serialized collision shapes (7,510 files). These contain the actual convex-hull / triangle-mesh colliders for bridges, buildings, walls, ramps, etc.

      Current state: Only terrain heightfield collision is active. You can drive on the ground but will pass through static objects.

      To enable prop collision: Reverse-engineer the Havok .cache serialization format (header + shape type enum + vertex/index arrays), parse each placement's Physics field into a Rapier/cannon-es collider, and add it to the physics world. This is Tier 3 per docs/vehicle-physics-port.md.
